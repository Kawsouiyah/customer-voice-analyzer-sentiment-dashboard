# Sentiment Analysis Dashboard

Analyse automatique des sentiments des retours clients avec VADER et Streamlit.

## Fonctionnalités
- Import de fichiers CSV
- Analyse en temps réel
- Visualisation interactive
- Export des résultats

## Technologies
- Python, Streamlit
- VADER, Pandas
- Plotly
